The Important Stuff

How it works:

The intraweb is made up of a lot of 'components'.' These components work together in unsion.
When we go to a web page the device we are using sends an HTTP request to the server for that page.
The server sifts through the ginormous amount of documents on the web to find the right one & sends
it back to the end user's browser (like IE) then sorts the information and makes it appear
on their screen.

HTML

HTML is an acronym that stands for HyperText Markup Language.
It is the common language of the internet. It contains text content that describes what is seen
and mark up that determines how the content looks.

Tags and Elements

The HTML documents are made up of elements. We use tags to tell the browser what type of elelment.
